# TASK MANAGER PROJECT

## A simple full-stack Task Manager application built using MERN Stack.

--------------------------------------------------
FEATURES
--------------------------------------------------

- User Register and Login
- JWT Authentication
- Create Tasks
- Update Tasks
- Delete Tasks
- Track Task Status
- Admin and User Roles
- Responsive UI

--------------------------------------------------
REQUIREMENTS
--------------------------------------------------

Install these before running the project:

1. Node.js
2. npm
3. MongoDB Atlas account or Local MongoDB
4. Git
5. VS Code (optional)

--------------------------------------------------
PROJECT SETUP
--------------------------------------------------

1. Clone Repository

git clone https://github.com/Ritwika-14/task-manager.git

2. Open Project Folder

cd task-manager

--------------------------------------------------
INSTALL DEPENDENCIES
--------------------------------------------------

Backend:

npm install

Frontend:

cd client
npm install
cd ..

--------------------------------------------------
ENVIRONMENT VARIABLES
--------------------------------------------------

Create a .env file in root folder and add:

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key

--------------------------------------------------
HOW TO RUN PROJECT
--------------------------------------------------

Start Backend:

npm start

Start Frontend:

cd client
npm run dev
